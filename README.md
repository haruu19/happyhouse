## :paperclip: 프로젝트 소개

- 주택정보를 모르는 지역에 출장이나 취업을 했을 때 지하철역 중심으로 출퇴근 가능한 거리의 주택정보를 제공하는 기능을 제공하려고 합니다.

## :hammer_and_pick: 개발스택

- SpringBoot 2.2.7.RELEASE
- JQuery
- MySQL
- Bootstrap
- Google Map, Place API
- Mybatis
